# Support

## Getting Help

For questions about using or contributing to the AI Quickstart Guide:

- Open a GitHub issue for bugs, broken links, unclear instructions, or content corrections.
- Use a discussion thread if discussions are enabled for broader implementation questions.
- For agency-specific customization, fork the repo and adapt it locally.

## What Maintainers Can Help With

Maintainers can usually help with:

- site build problems
- broken internal links
- stale source references
- unclear content
- deck-source formatting
- code-sample issues
- release artifact problems

Maintainers cannot provide:

- legal advice
- procurement approval
- security authorization for an agency deployment
- vendor selection decisions
- private implementation support for a specific agency unless separately arranged

## Before Filing an Issue

Please include:

- the page or file path
- what you expected
- what happened instead
- any command output, if this is a build or code-sample issue
- whether you are using the upstream repo or an agency fork

Do not include secrets, private data, or non-public agency records.
