# Deck Source: Quarterly Sponsor Report

## How to use this file

Use this as a deck companion to the quarterly milestone report. Replace all metrics, gates, risks, and asks with current program data.

## Deck instructions

- Audience: executive sponsor or steering group
- Meeting length: 15-30 minutes
- Desired deck length: 10 slides
- Tone: concise, factual, decision-oriented
- Use AI-generated images: optional; prefer simple charts from supplied data
- Do not invent metrics, savings, incidents, gate status, or asks

## Localization checklist

- Replace all quarter, metric, gate, spend, benefit, risk, ask, owner, and date placeholders.
- Verify finance, training, incident, use-case, and gate status data.
- Confirm sponsor reporting format and records requirements.
- Apply agency branding only after content and evidence review.

## Import prompt

Create a quarterly sponsor report deck. Follow slide breaks exactly. Use speaker notes as presenter notes. Do not invent metrics, savings, incidents, gate status, risks, or asks.
