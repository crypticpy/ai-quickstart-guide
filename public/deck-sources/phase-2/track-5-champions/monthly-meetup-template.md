# Deck Source: Track 5 Monthly Champions Meetup Template

## How to use this file

Use this reusable shell for monthly Champions Network meetings. Paste into an AI presentation tool and update the topic, prompts, and action items each month.

## Deck instructions

- Audience: AI Champions
- Meeting length: 90 minutes
- Desired deck length: 8 slides
- Tone: peer-oriented, concise, operational
- Do not invent tool approvals, metrics, or program decisions

## Import prompt

Create a reusable monthly meetup deck. Follow slide breaks exactly. Use speaker notes as presenter notes. Preserve placeholders.
