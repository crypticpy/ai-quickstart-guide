# Deck Source: Track 6 Monthly Clinic Template

## How to use this file

Use this reusable shell for monthly role-family AI clinics. Update the role family, examples, and actions each month.

## Deck instructions

- Audience: domain experts from one role family
- Clinic length: 90 minutes
- Desired deck length: 8 slides
- Tone: applied, reflective, domain-led
- Do not invent examples, errors, metrics, or domain conclusions

## Import prompt

Create a reusable monthly clinic deck. Follow slide breaks exactly. Use speaker notes as presenter notes. Preserve placeholders.
