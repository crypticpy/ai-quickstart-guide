## How to use this file

Use this short lab support deck to frame a role-specific applied lab. Replace all examples with governance-approved, sanitized outputs for the target role family. Do not use it to teach the domain.

## Deck instructions

- Audience: domain experts in one role family
- Lab length: 3 hours
- Desired deck length: 9 slides
- Tone: domain-deferential, practical, rigorous
- Do not invent domain rules, case facts, statutes, or evaluation results
- The senior domain facilitator should own correctness judgments
