# Deck Source: Track 6 Domain Lab 2 - Workflow Integration

## How to use this file

Use this short lab support deck to frame a role-specific lab on where AI fits in real work. Replace examples with local role-family workflows and approved tools. Lab 2 should happen only after participants have tried an approved tool on authorized low-stakes work.

## Deck instructions

- Audience: domain experts in one role family
- Lab length: 3 hours
- Desired deck length: 9 slides
- Tone: domain-deferential, applied, realistic
- Use only approved tools and low-stakes work authorized for the role family
- Do not invent workflow facts, tool approvals, or job changes
