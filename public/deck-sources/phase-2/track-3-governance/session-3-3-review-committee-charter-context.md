# Deck Source: Track 3 Session 3.3 - Review Committee Charter Context

## How to use this file

Use this as a short framing deck before drafting the Review Committee charter and review checklist.

## Deck instructions

- Audience: governance reviewers and program leadership
- Session length: 90 minutes
- Deck length: 7 slides
- Purpose: orient the room, then draft the charter
- Do not invent committee authority, quorum, records obligations, or agency facts

## Import prompt

Create a short context-setting deck. Follow slide breaks exactly. Use speaker notes as presenter notes.
