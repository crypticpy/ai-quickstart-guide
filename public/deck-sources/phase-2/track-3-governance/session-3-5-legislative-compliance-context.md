# Deck Source: Track 3 Session 3.5 - Legislative Compliance Context

## How to use this file

Use this as a short framing deck before building the agency's AI legislative compliance matrix. Verify all legal references against current official sources and local counsel before use.

## Deck instructions

- Audience: legal, compliance, policy, privacy, procurement, security, and program leads
- Session length: 90 minutes
- Deck length: 7 slides
- Purpose: orient the room, then work on the compliance matrix
- Do not invent laws, effective dates, obligations, or legal conclusions

## Import prompt

Create a short context-setting deck. Follow slide breaks exactly. Use speaker notes as presenter notes. Do not invent laws, effective dates, legal obligations, or compliance conclusions.
