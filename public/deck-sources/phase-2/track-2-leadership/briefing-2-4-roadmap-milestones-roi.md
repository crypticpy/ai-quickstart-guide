# Deck Source: Track 2 Briefing 2.4 - Roadmap, Milestones, And ROI

## How to use this file

Paste or upload this Markdown into an AI presentation tool and ask it to create a first-draft executive briefing deck. Localize roadmap, milestone gates, ROI, and starter candidates before use.

## Deck instructions

- Audience: executive leaders and program sponsors
- Briefing length: 60 minutes
- Desired deck length: 13 slides
- Tone: practical, strategic, decision-oriented
- Decision prompt: Which starter project do we fund first?
- Do not invent ROI, project readiness, budget approvals, or agency facts
