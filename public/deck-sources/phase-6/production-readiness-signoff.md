# Deck Source: Production Readiness Sign-Off

## How to use this file

Use this deck for a go/no-go meeting before launching a starter AI project. Replace all criteria, evidence, risks, and sign-off roles with local project data.

## Deck instructions

- Audience: product owner, sponsor, Review Committee, security, privacy, operations, support
- Meeting length: 45-60 minutes
- Desired deck length: 11 slides
- Tone: operational, evidence-based, decision-oriented
- Use AI-generated images: optional; avoid visuals that imply approval before sign-off
- Do not invent readiness evidence, approvals, incidents, or metrics

## Localization checklist

- Replace all project, owner, launch, evidence, condition, and sign-off placeholders.
- Confirm governance, security, privacy, records, procurement, accessibility, support, and operations evidence.
- Attach or link actual evidence in backup materials before the meeting.
- Apply agency branding only after content and evidence review.

## Import prompt

Create a production readiness sign-off deck. Follow slide breaks exactly. Use speaker notes as presenter notes. Do not invent evidence, approvals, incidents, metrics, or agency facts.
