# Lab 4.5 starter

This directory contains a small intentionally flawed Python codebase. Use an AI coding assistant (Claude Code, GitHub Copilot, Cursor, or Continue.dev) to work through the five tasks in `tasks.md`.

## Setup

```bash
cd code-samples/track-4/lab-5/starter/sample-app
python3.12 -m venv .venv
source .venv/bin/activate
pip install -e ".[dev]"
pytest -q
```

You should see one passing test.

## Order of work

Read `tasks.md` from top to bottom. Each task takes about 25 minutes. After each task, run `pytest -q`.

## What you keep

Your committed diff is the deliverable. Treat AI suggestions as drafts. Read every line before you accept it.

The reference shape lives in `../solution/`. Try a task before peeking.
