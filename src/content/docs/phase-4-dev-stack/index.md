## Objective

Select (or validate) the agency's development stack and establish the coding standards, testing practices, and developer workflows that will govern all platform and application development.

## Deliverables

- Stack selection decision tree (Python/React, .NET, Java/Spring, Node/Next, Go)
- RAD reference implementation walkthrough (Python / TypeScript / FastAPI / React)
- Coding standards (linting, formatting, naming, commit messages, PR review)
- AI-assisted development workflow (Copilot, Claude Code, Cursor — gov-safe configuration)
- Testing strategy (unit, integration, contract via Pact, E2E)
- API-first design standards (OpenAPI specs, versioning, contract testing)
- ADR template and workflow

## Hard dependency

Requires Phase 3 (infrastructure) to be operational — coding standards need a working test/deploy pipeline to enforce.
