All notable changes to the AI Quickstart Guide are tracked here.

This project uses semantic versioning for public releases. Until `v1.0.0`, minor releases may still adjust page structure, but changes that affect forks will be called out here.

## [0.1.0] - 2026-05-01

### Added

- Initial public preview of the AI Quickstart Guide site.
- Six-phase AI adoption playbook with small, standard, and large agency paths.
- Interactive tools for readiness, AUP drafting, risk tiering, use-case intake, ROI, compliance filtering, and starter-project selection.
- Track 4 developer code labs with starter and solution directories.
- AI-importable presentation deck-source library.
- Offline playbook export workflow for Markdown, PDF, DOCX, and release pack artifacts.

### Notes

- This is a public preview. Agencies should localize policy, procurement, labor, security, and legal references before operational use.
